Generate, review, and compile AI-driven layouts to YOOtheme Pro JSON inside WordPress with direct YOOtheme customizer integration.

== Description ==

AI Layout for YOOtheme is a powerful WordPress plugin that integrates directly with YOOtheme's customizer to provide AI-powered layout generation. Create professional layouts using natural language descriptions and OpenAI's advanced AI models.

= Key Features =

* **YOOtheme Extension Integration**: Seamlessly integrated with YOOtheme customizer
* **Vue.js Interface**: Modern, responsive UI built with Vue 3
* **AI-Powered Generation**: Create layouts using OpenAI's GPT models
* **Real-time Compilation**: Convert AI descriptions to YOOtheme Pro JSON
* **Direct Customizer Access**: No need to leave the YOOtheme interface
* **Professional Build System**: Webpack-based extension architecture

= How It Works =

1. **Activate the plugin** in WordPress admin
2. **Open YOOtheme customizer** on any page
3. **Navigate to "AI Layout" panel** in the sidebar
4. **Describe your desired layout** in natural language
5. **Generate and review** AI-created layouts
6. **Download or apply** layouts directly to your site

= Use Cases =

* **Web Designers**: Quickly prototype layouts for clients
* **Content Creators**: Generate landing page layouts
* **Developers**: Speed up YOOtheme development workflow
* **Agencies**: Create consistent design systems

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-layout-for-yootheme` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Open YOOtheme customizer to access the AI Layout panel
4. Configure your OpenAI API key in the plugin settings

== Frequently Asked Questions ==

= Do I need YOOtheme Pro? =

While the plugin works with YOOtheme themes, YOOtheme Pro provides the best experience with the customizer integration.

= What OpenAI models are supported? =

The plugin uses OpenAI's latest GPT models for layout generation and analysis.

= Can I customize the generated layouts? =

Yes! All generated layouts are fully customizable within YOOtheme's interface.

= Is my data secure? =

Yes, the plugin follows WordPress security best practices and only sends necessary data to OpenAI.
